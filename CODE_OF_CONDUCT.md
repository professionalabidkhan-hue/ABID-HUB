# Code of Conduct

## Our Pledge
We pledge to create a welcoming, professional, and inclusive environment for all contributors, students, and collaborators.

## Standards
Examples of positive behavior:
- Using respectful and inclusive language
- Providing constructive feedback
- Focusing on clarity, reproducibility, and professionalism

Examples of unacceptable behavior:
- Harassment or discriminatory language
- Publishing sensitive or private information
- Disrespecting privacy and compliance standards

## Responsibilities
- Project maintainers will enforce this Code of Conduct.
- Contributors are expected to uphold these standards in all interactions.

## Enforcement
Instances of unacceptable behavior may be reported to the project maintainers.  
All complaints will be reviewed and addressed fairly.

## Scope
This Code of Conduct applies to all project spaces, including GitHub Issues, Pull Requests, and discussions.
